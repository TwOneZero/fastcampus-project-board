# 게시판 서비스 - 패스트캠퍼스, 10개 프로젝트로 완성하는 백엔드 웹개발




가장 기본적이고 보편적인 게시판 기능
스프링 부트 버전 2.7 관련 기술들, 자바 17 기능들, 개발 도구들을 경험을 위한 프로젝트

이 서비스는 패스트캠퍼스의 10개 프로젝트로 완성하는 백엔드 웹개발(Java/Spring) 초격차 패키지 Online 강의를 참고하여 만들었습니다.
